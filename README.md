# Texas Coastal Water Quality Analysis

## Project Overview

This project analyzes Texas coastal water quality data to identify temporal trends, geographic patterns, and potential environmental concerns.

The analysis will demonstrate practical skills in data cleaning, validation, exploratory data analysis, statistical modeling, SQL, and data visualization.

## Research Questions

1. How does water quality vary across Texas coastal monitoring locations?
2. Are there seasonal or long-term trends in temperature, salinity, dissolved oxygen, or other indicators?
3. Which monitoring locations show unusual values or potential data quality issues?
4. Are environmental variables significantly associated with one another?

## Tools and Technologies

- Python
- pandas
- NumPy
- Matplotlib
- SQL
- Jupyter Notebook
- Statistical analysis

## Planned Workflow

1. Collect public Texas coastal water quality data
2. Inspect and document the dataset
3. Clean and validate the records
4. Analyze temporal and geographic patterns
5. Create clear visualizations
6. Conduct statistical analysis
7. Summarize findings and limitations
